Sometimes we prefer to hide our hard disk drive to be listed in My Computer. It's a good idea to store all your personal data in any single partition of hard disk and then hide that partition. Although, hiding this way won't stop the smart computer users to access those hidden partitions. But still this works for general users and children.

Hiding hard disk partition in Windows requires a minor tweak into the registry keys. Follow the below easy steps to hide your hard partition.
  • Open Run menu
  • Type gpedit.msc in Run menu and click OK. It will open up Group Policy window.
  • Navigate to the following template in the Local Group Policy Editor:
User Configuration > Administrative Templates > Windows Component > Windows Explorer
  • Locate the template Hide these specified drives in My Computer and double-click it.
Double click on "Hide these specified drives in My Computer" template
  • Select Enabled from Not Configured and select the combinations from drop down menu to hide the partition(s).
Select "Enabled" and Drive Combinations to hide
Restart the system to see the changes you have made. At anytime, you can revert these changes by disabling settings in the Group Policy Editor.
