The Google Drive team has officially introduced the all new Save To Drive extension and added zooming and commenting capabilities to images. You can install the browser extension from the Chrome Web Store while you can try the image zooming and commenting features directly on your Google Drive account.

Let's take a glimpse on chrome extension first. Once you are done with the installation, you can click the Save To Drive extension icon located beside the address bar to save an image of entire/visible page, the HTML page, or a web archive (.mht). You can automatically convert Microsoft Office files or comma separated files to Google docs format. Optionally you can right click on images or links to save them directly to Drive.

Once the image is saved in the Drive you are prompted to rename/trash it, if you wish. The default location of saved media is the root folder of your account. Currently there is no option to select existing folders from your Drive to save the media within.

Google has also added zooming feature to the images. Now you can zoom your pictures stored in Drive by scrolling or using the new "fit to page" and 100% button. And if you have something to say about a specific part of an image the you can select that specific area and add a comment to it! Isn't it great!

Both the features are small yet significant. Google team continuously evolving its each and every services from good to great! This extension will probably add any of the following features in the coming days.
  • Location/Folder selection before saving media
  • Sharing options once media saved to Drive
