There are ample number of PDF Tools and Services are available on internet that can perform PDF conversions and creations. However, they are of no help when you wish to create PDF from some session based webpages like online shopping receipts or transaction pages on banking websites. Indeed, you don't need any PDF services or extensions as long as you've installed Google Chrome browser!

Open the web page you want to save as PDF in Chrome and press Ctrl + P (Cmd + P on Mac). It will open up the Print Dialog. Change the print destination to "Save as PDF" and press "Save".


"Save as PDF" options in Windows version of Chrome
Optionally you can print the selective pages or page ranges (like 1-5, to print 1st five pages only), set page headers, layout of the page to either Portrait or Landscape, etc.

You don't need to install any fancy extensions or desktop software as far as you've installed Chrome on your PC. Chrome itself works as PDF Creator. You can also convert your locally stored documents and images into PDF format by opening them in Chrome.
