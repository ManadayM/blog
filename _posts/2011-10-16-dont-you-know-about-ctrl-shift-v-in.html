It might happened, you are writing an email and want to copy some excerpt from webpage in the mail content. But when you paste the text, alas, all the source formatting like colors, spacing, font style everything also get copied!!!

This may happen to us so many times but fortunately Chrome provides a workaround to this problem alongside the simple ctrl + v keyboard shortcut for "paste", Chrome supports a similar shortcut ctrl + shift + v for "paste as plain text". (On mac os, it is cmd + shift + v).

You can use this shortcut in any richtext editor (like Gmail's mail composing window or writing in google docs) to paste the text without any source formatting.

Source text, ctrl + c

Paste, ctrl + v

Paste as plain text, ctrl + shift + v
