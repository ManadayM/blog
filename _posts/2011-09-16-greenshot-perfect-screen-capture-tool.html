Print screen (Alt + Print screen, too) is always boring task in windows XP, you've to instantly paste your screenshot image to either word document file or Microsoft paint to save it.Windows XP also lacks region screen capturing facility.

We can get rid off this problem. There are so many software available but some of them are paid tools. Greenshot is a free screen capturing tool. It is quite good and useful for those who are doing screen capturing for their daily documentation work.
