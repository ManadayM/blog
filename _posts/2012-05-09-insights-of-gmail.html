Gmail, one of the best web mail service by Google. Many of you might be using it regularly. But, many of times I've seen that user don't utilize all the features that are supported by Gmail! This post links to the few posts that describes some useful features in-depth.

Let's have some glimpses on Gmail, first.

Gmail at a glance

It's a free email service by Google. It was launched on April 1, 2004 as an invitation only beta release with limited features and it became publicly available on February 7, 2007.

Gmail significantly increased the webmail standards by giving 1Gb storage capacity per user, from the 2 to 4Mb its competitors such as hotmail offered at that time. You can send 25Mb of message, including attachments, in a single shot, which is much higher as compared to other mail service providers. As of now, Gmail provides 10Gb of storage space per user. Which was 7Gb few days ago!

Features

Gmail gives access to tons of cool features and many more as optional in Gmail Labs. I will keep posting on gmail features as different articles and link them here as a list rather writing all about gmail in a single go.
  • Keyboard shortcuts
  • Offline support
  • Plus addressing in your gmail id
