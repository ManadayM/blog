Are you running low on space in your Gmail account? If so, calm down then. Lately Gmail has introduced much helpful search operator called Size that will enable you to search all those hefty mails that are occupying sizable space of your gmail account. Once you are done with searching the space hogging mails then you can decide the next action (keep, delete, forward or save in Google Drive and delete).

Search large emails using "size" operator in Gmail
Size operator requires you to provide message size either in Bytes (approx. 1000000 Bytes = 1 MB) or in Mega Bytes (MB). It will search and list down for the mentioned sized or larger than that messages. In the above screenshot, for example, I got search result for the messages whose size is 8MB or more than that.

That's it. You can also use other Gmail search operators with size operator to refine results like has:attachment to list out only emails with attachment.
